## ESM3的预训练模型有下列几个？
```
# Model names  
ESM3_OPEN_SMALL = "esm3_sm_open_v1"  
ESM3_STRUCTURE_ENCODER_V0 = "esm3_structure_encoder_v0"  
ESM3_STRUCTURE_DECODER_V0 = "esm3_structure_decoder_v0"  
ESM3_FUNCTION_DECODER_V0 = "esm3_function_decoder_v0"
```
### ESM3的预训练模型包括以下几个：

1. **ESM3_OPEN_SMALL**: "esm3_sm_open_v1" - 这可能是一个较小版本的ESM3模型，专为开放使用而设计。  
2. **ESM3_STRUCTURE_ENCODER_V0**: "esm3_structure_encoder_v0" - 这是结构编码器的一个预训练版本，用于编码蛋白质的结构信息。 
3. **ESM3_STRUCTURE_DECODER_V0**: "esm3_structure_decoder_v0" - 这是结构解码器的一个预训练版本，用于从编码的结构信息中恢复或预测蛋白质结构。  
4. **ESM3_FUNCTION_DECODER_V0**: "esm3_function_decoder_v0" - 这是功能解码器的一个预训练版本，用于解析蛋白质的功能特征或功能区域。  

这些模型可以被用于各种生物信息学和结构生物学的应用，包括但不限于结构预测、功能注释和相互作用预测。
