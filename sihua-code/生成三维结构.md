## 生成三维结构 
### 总结：
这段代码演示了如何使用ESM3模型，一种基于转换器的蛋白质结构和序列预测模型，进行蛋白质设计和结构预测。该流程包括：  

使用用户的API密钥登录Hugging Face Hub。  
下载和实例化一个预训练的ESM3模型，并将其配置到适当的计算设备上。  
使用模型来填充一个部分已知的蛋白质序列，生成完整的蛋白质序列。  
根据完整的蛋白质序列预测其三维结构。  
将预测的结构保存为PDB文件。  
进行反向设计，即通过已知结构预测新的蛋白质序列，再基于新序列预测结构，完成从结构到序列再到结构的循环设计。这种方法在药物设计和蛋白质工程中非常有用。  
```
# 从 huggingface_hub 和 esm (进化尺度建模) 库中导入所需的类和函数。
from huggingface_hub import login
from esm.models.esm3 import ESM3
from esm.sdk.api import ESM3InferenceClient, ESMProtein, GenerationConfig

# 启动登录过程以使用 Hugging Face Hub 的 API 密钥进行用户认证。
login()

# 使用指定的预训练模型实例化一个 ESM3 模型对象，并将模型转移到 CUDA 设备上以利用 GPU 加速。
model: ESM3InferenceClient = ESM3.from_pretrained("esm3_sm_open_v1").to("cuda")  # 或者使用 "cpu" 在 CPU 上运行。

# 定义蛋白质碳酸酐酶的部分序列，缺失的区域由下划线表示。
prompt = "___________________________________________________DQATSLRILNNGHAFNVEFDDSQDKAVLKGGPLDGTYRLIQFHFHWGSLDGQGSEHTVDKKKYAAELHLVHWNTKYGDFGKAVQQPDGLAVLGIFLKVGSAKPGLQKVVDVLDSIKTKGKSADFTNFDPRGLLPESLDYWTYPGSLTTPP___________________________________________________________"

# 将蛋白质序列包装在一个 ESMProtein 对象中。
protein = ESMProtein(sequence=prompt)

# 使用指定的配置参数生成序列中缺失的部分。这个阶段完成蛋白质序列的生成。
protein = model.generate(protein, GenerationConfig(track="sequence", num_steps=8, temperature=0.7))

# 基于新完成的序列生成蛋白质的三维结构。
protein = model.generate(protein, GenerationConfig(track="structure", num_steps=8))

# 将预测的结构保存为 PDB 文件。
protein.to_pdb("./generation.pdb")

# 重置序列信息，启动一个新的设计周期，通过结构反向工程序列。
protein.sequence = None
protein = model.generate(protein, GenerationConfig(track="sequence", num_steps=8))

# 重置结构坐标，以便从重新设计的序列生成新结构。
protein.coordinates = None
protein = model.generate(protein, GenerationConfig(track="structure", num_steps=8))

# 将从重新设计的序列预测的新结构保存到另一个 PDB 文件中。
protein.to_pdb("./round_tripped.pdb")

```
